<h>核心价值</h>
- 最小可行性产品
> 通过图像识别和文字识别技术，对被拍摄的事物进行识别，并进行解释或科普
> 通过本产品帮助使用者快速认识到眼前的事物，并对其有一个大概的了解
> 本产品还将致力于服务视力障碍人群，提供一个认识新事物的平台
